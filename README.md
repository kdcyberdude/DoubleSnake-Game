# DoubleSnake-Game
Two Player Snake Game on Same Computer.<br />
First Snake Controls -<br />
  W: UP<br />
  D: Right<br />
  S: Down<br />
  A: Left<br />
Second Snake Controls -<br />
  up-arrow:<br />
  down-arrow:<br />
  left-arrow:<br />
  right-arrow:<br />
  <br />
Fun Fact: Try to play with two hands... It's really challenging!!
